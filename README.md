# FUTURE_PE_02 — AI Content Marketing using UGC Ads

## Overview
This repository contains Task 2 of the Future Interns Prompt Engineering internship: generating a complete UGC-style ad content pack for a local business using AI.

## Details
- **Business:** Manish Gentora Cafe, Erode
- **Tool Used:** Claude AI
- **Deliverable:** UGC-style ad scripts, scroll-stopping hooks, problem-solution-CTA ads, and platform-adapted versions (Instagram, Facebook, YouTube Shorts)

## Contents
- `prompts/` — all prompts used
- `outputs/` — all AI-generated ad content
- `docs/Prompt_Engineering_Report_Task2.docx` — full task report
- `screenshots/` — process screenshots

## Skills Demonstrated
- AI-driven content marketing
- UGC ad scripting and hooks
- Prompt engineering for conversions
- Brand voice adaptation across platforms

## Key Learnings
- Explicit tone constraints are essential for authentic-sounding UGC content.
- Naming specific hook styles produces more varied, usable outputs than generic requests.
- Platform adaptation works best as a two-step process: one strong base script, then tailored reformatting.
